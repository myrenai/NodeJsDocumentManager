My Document Manager
===
My Document Manager is an app based on node.js and AngularJS. 

## Installation
```bash
$git clone https://github.com/jiyoungParkKim/myDocumentManager.git
$cd  myDocumentManager
$npm install
$bower install
$npm start
```
## Reference

 * ace editor: https://github.com/ajaxorg/ace
